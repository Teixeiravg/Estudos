QUESTIONÁRIO

Questão1: Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo.
Resposta: Entrega arquivos estáticos como HTML, CSS e imagens. Ex: Apache.

Questão2: Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?
Resposta: Leve e eficiente. Consegue lidar com milhares de conexões simultâneas com baixo consumo de recursos e alta confiabilidade.

Questão3: Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?
Resposta: HTTP: sem criptografia.
HTTPS: usa criptografia SSL/TLS.
HTTPS é mais seguro, protege os dados transmitidos.

Questão4: Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?
Resposta: Portas são canais de comunicação. A porta 443 é usada pelo HTTPS, essencial para segurança e privacidade na web.

Questão5: Defina o conceito de endereço IP e explique sua importância em redes de computadores.
Resposta: É um número que identifica dispositivos na rede. Permite comunicação, segurança e rastreabilidade.

Questão6: Explique o que é hospedagem VPS e em quais situações ela é mais indicada.
Resposta: Servidor virtual com mais controle e desempenho. Ideal para projetos com tráfego médio a alto, e-commerce e múltiplos sites.

Questão7: Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.
Resposta: Configuração simples, HTTPS automático e gratuito, alta performance e foco em segurança.

Questão8: Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.
Resposta: ASP.NET Core. Suporta totalmente C# e oferece ferramentas integradas para desenvolvimento.

Questão9: Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.
Resposta: O cliente (navegador) faz a requisição, o servidor (web) responde com a página ou dados.

Questão10: Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.
Resposta: Servidor de aplicação usado em aplicações Java (Servlet e JSP).

Questão11: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.
Resposta: Servidor web entrega arquivos estáticos. Servidor de aplicação executa lógica de programação para gerar conteúdo dinâmico.

Questão12: Defina o conceito de domínio na internet e sua relação com os endereços IP.
Resposta: Domínio é um nome legível (ex: google.com) que o DNS traduz para o endereço IP correspondente do servidor.

Questão13: Quais as principais vantagens da hospedagem em nuvem em comparação com a hospedagem local?
Resposta: Escalabilidade, alta disponibilidade, menor custo com infraestrutura e acesso remoto facilitado.

Questão14: Cite dois exemplos de servidores de aplicação e as linguagens/plataformas que eles suportam.
Resposta: WildFly: Java EE.
Tomcat: Java Servlet e JSP.

Questão15: Por que é essencial a utilização de HTTPS em sites e sistemas atualmente? Quais riscos o HTTP apresenta?
Resposta: HTTPS criptografa os dados e garante segurança. HTTP permite interceptação, roubo e modificação de dados.

Questão16: Dê um exemplo de situação onde o uso de hospedagem VPS seria mais vantajoso do que uma hospedagem compartilhada.
Resposta: Site em crescimento com aumento de tráfego, que precisa de controle e personalização no servidor.

Questão17: Diferencie os conceitos de IP e porta em uma rede, explicando suas funções.
Resposta: IP identifica o dispositivo na rede. Porta identifica o serviço ou aplicação no dispositivo.

Questão18: Qual a importância de um servidor web conseguir lidar com conexões via HTTPS?
Resposta: Garante comunicação segura e protegida contra ataques, aumentando a confiança do usuário.

Questão19: Cite dois exemplos de servidores web e explique suas principais características.
Resposta: NGINX: alto desempenho, proxy reverso, arquitetura assíncrona.
Apache: modular, flexível e amplamente suportado.

Questão20: Defina o que é uma hospedagem compartilhada e em quais casos ela costuma ser utilizada.
Resposta: Vários sites compartilham os recursos do mesmo servidor. Ideal para sites pequenos e de baixo custo.

Questão21: Explique a função de um servidor de aplicação e a diferença em relação a servidores que apenas servem arquivos estáticos.
Resposta: Processa lógica da aplicação e gera conteúdo dinâmico. Servidor estático apenas entrega arquivos prontos.

Questão22: Cite um servidor de aplicação usado para projetos em Python e descreva como ele funciona.
Resposta: Gunicorn: servidor WSGI que executa aplicações Python com múltiplos processos para alta performance.

Questão23: O que é o WildFly (ex-JBoss) e para qual tecnologia ele é indicado?
Resposta: Servidor de aplicação Java EE, usado para aplicações empresariais em Java.

Questão24: Qual a função da porta 21 em redes de computadores e para qual serviço ela é utilizada?
Resposta: Porta do protocolo FTP, usada para controle da transferência de arquivos.

Questão25: Explique por que o servidor LiteSpeed é compatível com o Apache e quais as vantagens dessa compatibilidade.
Resposta: Suporta configurações do Apache (.htaccess), facilitando migração com melhor desempenho e menor uso de recursos.

Questão26: O que é o Microsoft IIS e em quais situações ele costuma ser adotado?
Resposta: Servidor web da Microsoft, usado em ambientes Windows com aplicações ASP.NET.

Questão27: Explique para que serve o Node.js e em quais tipos de aplicações ele é mais utilizado.
Resposta: Executa JavaScript no servidor. Usado em aplicações em tempo real, APIs e alto volume de conexões simultâneas.

Questão28: Descreva a função do servidor GlassFish e para qual linguagem ele é destinado.
Resposta: Servidor de aplicação Java EE para aplicações empresariais em Java.

Questão29: O que são Puma e Unicorn e em quais projetos eles são normalmente utilizados?
Resposta: Servidores de aplicação para Ruby on Rails. Gerenciam múltiplos processos para alta performance.

Questão30: Qual a importância da escolha adequada entre hospedagem local, compartilhada, VPS e nuvem no planejamento de um projeto web?
Resposta: Afeta desempenho, custo, segurança e escalabilidade. Influencia diretamente no sucesso e sustentabilidade do projeto.
